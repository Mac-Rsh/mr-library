`mr-library`
# 简介
**mr-library** 创建于2023年1月，是一款开源驱动库。mr-library主要采用C语言编写，可快速移植到各种平台（主要以嵌入式mcu为主）。

**mr-library** 区分驱动内核、驱动抽象层、设备层，都支持模块化裁剪。驱动抽象层和设备层支持独立使用。

----------

# **mr-library** 特点
* 资源占用低。
- 简单易用，结构清晰。
- 通用性强，不同平台仅需适配少量接口即可使用。
* 松解耦，模块化。

 ----------

# 代码目录
mr-library源代码目录结构如下图所示：
|名称|描述|
|:---|:---|
|bsp|板级支持包|
|device|设备层文件|
|driver|驱动抽象层文件|
|include|内核头文件|
